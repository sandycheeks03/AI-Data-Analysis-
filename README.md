## Project Title
**AI-Powered Data Analyzer with Visualization**

## Description
This project analyzes a dataset using Python libraries to extract insights, perform statistical analysis, and create visualizations. The analyzer cleans data, computes descriptive statistics, and generates meaningful charts to help understand patterns in the data.

## Python Libraries Used
- **pandas**: Data manipulation and analysis
- **numpy**: Numerical computations
- **matplotlib**: Data visualization
- **seaborn**: Advanced statistical visualizations

## How to Run the Code
1. Clone this repository:
```bash
   git clone https://github.com/sandycheeks03/AI_Programming_Project.git
   cd AI_Programming_Project
```

2. Install required libraries:
```bash
   pip install pandas numpy matplotlib seaborn
```

3. Run the analyzer:
```bash
   python data_analyzer.py
```
